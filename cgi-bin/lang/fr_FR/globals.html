Cette interface est utilisée pour la définition des paramètres globaux de SquidGuard.

Répertoire de log:

	Ce paramètre obligatoires permet de définir le répertoire de base dans
	lequel seront sauvegardés les logs.
	Par défaut: /usr/local/squidGuard/logs

Répertoire des bases:

	Ce paramètre obligatoires permet de définir le chemin complêt vers le
	répertoire de stockage des base de données de listes.
	Par défaut: /usr/local/squidGuard/db

Base de données MySQL:

	Si votre version de SquidGuard est supérieure ou égale à 1.4 vous pouvez
	utiliser une base de données MySQL pour rechercher des utilisateurs lors
	de la définition de vos sources. Si vous utiliser cette possibilité vous
	devez definir les différents paramètres  de connexion a une base MySQL
	dans la section "Extraction des utilisateurs depuis MySQL".

Base de données LDAP:

	Si votre version de SquidGuard est supérieure ou égale à 1.4 vous pouvez
	utiliser une base de données LDAP pour rechercher des utilisateurs lors
	de la définition de vos sources. Si vous utiliser cette possibilité vous
	devez definir les différents paramètres  de connexion a une base LDAP
	dans la section "Extraction des utilisateurs depuis LDAP".
