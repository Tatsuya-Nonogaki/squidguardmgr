Ici sont affichées toutes les listes définies dans le répertoire dbhome de SquidGuard.

Notez que si vous avez défini des sous répertoires dans celui-ci pour sur classifier vos 'blacklists', SquidGuard Manager ne fonctionnera pas correctement et risque de corrompre votre fichier squidguard.conf!

Notez aussi que si vous avez défini des liens symboliques dans ce répertoire, SquidGuard Manager ne fonctionnera pas correctement et ils ne seront pas affichés ici.

En cliquant sur le nom d'une liste, vous aurez la possibilité d'ajouter/supprimer des entrées dans les fichiers de liste de domaines, d'urls et d'expressions. Un lien est aussi à disposition pour éditer directement ces fichiers.

Le bouton Reconstruire toutes les bases de données vous permet de lancer la reconstruction des bases Berkeley de toutes les listes. Tout commme la commande : "squidguard -C all"

Il peut être difficile de savoir dans quelle liste une URL rejetée se trouve. Utilisez le formulaire de recherche pour cela.

Pour créer une nouvelle liste, cliquez sur le bouton Créer une nouvelle liste et entrez le nom de cette liste. Le nom ne doit comporter que des caractères alphanumériques et les caractères tiret et underscore '-', '_'.

Pour supprimer une liste cliquez sur l'icone de suppression à coté de son nom de liste.
