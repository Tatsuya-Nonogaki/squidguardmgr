Les champs Alias et Description ci-dessus peuvent être utilisé pour traduire et décrire les différentes listes pour une meilleur compréhension de l'interface utilisateur.

Domaines : n'importe quelle chaine de charactere qui peut etre trouvée dans la partie 'hostname' de l'URL
URLs : n'importe quelle chaine de charactere qui peut etre trouvée dans la partie 'hostname' et URI de l'URL
Expressions : n'importe quelle expression regex dont le pattern peut s'appliqué à une URL

Ceci est une interface simplifiée pou l'ajout ou la suppression d'enregistrements dans les fichiers de liste des catégories ou filtres.

Simplement saisissez vos entrées dans le champ texte de un ou plusieurs type de blacklist et cliquez sur le bouton Ajouter pour insérer ces nouvelles entrées ou cliquez sur le bouton Supprimer pour les oter des fichiers concernés.

Vous pouvez ajouter/supprimer plusieurs entrées par type de liste en les saisissant ligne par ligne dans les zone de texte.

Si vous voulez éditer directement un fichier de l'un de ces types de liste, cliquez sur l'icone Editer de la liste correspondante.

Après la dernière modification dans ces listes vous devez cliquer sur l'icone Reconstruire la base de données pour activer les changements.
