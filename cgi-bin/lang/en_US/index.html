Globals:
		This menu allow you to define SquidGuard globals variables.

	Schedules:
		This menu allow you to define schedules time to be used in other SquidGuard object (Sources, URL Rewrites, Filters and ACLs).

	Sources:
		Use this menu to define sources like users, groups, Ip addresses, range of Ip addresses or networks to apply policies on them.

	Url rewriting:
		This menu allow you to define as many as URL rewriting rules you want that will be applied by SquidGuard on pattern matching.

	Filters:
		This menu allow you to defined URL destination groups based on the predefined lists of domain, url or regexp expressions.

	ACLs:
		This is the central place where you defined your access rules to apply to the different source object following the defined filter destination

	View Configuration:
		This menu allow you to see the SquidGuard configuration file.

	Manage Lists:
		This menu allow you to manage all lists by editing them fully or adding/removing entries.

	Logs:
		This menu allow you to consult all log files you'd defined in the SquidGuard configuration.

	SquidClamav:
		With this link you will display a similar interface to administer SquidClamav configuration file. This menu is enable only if you have SquidClamav installed on the system.

	Restart Squid
		This menu allow you to 'reconfigure' Squid when all changes to SquidGuard and/or SquidClamav redirectors configuration files are done.
