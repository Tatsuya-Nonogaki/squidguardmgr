Here are all the lists defined in the SquidGuard dbhome directory.

Note that if you have created subdirectory in this dbhome directory to subclassify your blocklists, SquidGuard Manager will not work and may corrupt your squidguard.conf file!

Note also that if you have defined symbolic links into that directory SquidGuard Manager will not work and they will not be displyed here

By clicking on a list name you will be allowed to add/remove entries from the domain, url and expression files of this list. You will also have a link to edit directly those files.

The Rebuild all databases button allow you to reconstruct all Berkeley database of the whole lists. It's just like a "squidguard -C all"

Some time it is difficult to know where a blacklisted domain or url is stored. Use the search form to know in witch list it appears.

To add a new list, click on the Create a list button and give it a name. The name must only use alphanumeric, dash '-' and underscore '_' characters.

To delete a list, just click on the icon attached to each list name.
