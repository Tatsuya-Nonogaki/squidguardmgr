This is a simple interface to add or remove entries in the blacklist files of a category or filter.

Just type your entries in the textarea for each type of blacklist and click on the Remove button if you want to remove it or click on the Add button to add them to the list.

You can add or remove multiple entries per type of list by typing them line by line in the textaera of the lists.

If you want to edit the list file directly, click on the Edit icon at the left of each textarea.

Once you finished modification of these lists, you must click on the Rebuild database icon to activate the changes.
