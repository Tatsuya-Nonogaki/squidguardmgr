This interface is used to define global parameters of SquidGuard.

Log directory:

	This mandatory parameter is used to define the full path where logi
	files will be written.
	By default: /usr/local/squidGuard/logs

Databases home:

	This mandatory parameter is used to define the full path of the base
	directory where the blacklist databases are stored.
	By default: /usr/local/squidGuard/db

MySQL Database:

	If you have a SquidGuard version above or equal to 1.4, you can use a
	mySQL database to find users in a source definition. If you plan to
	do that, use the settings in the "Retrieving users from MySQL" part to
	set the database connection parameters.

LDAP Database:

	If you have a SquidGuard version above or equal to 1.5, you can use a
	LDAP database to find users in a source definition. If you plan to
	do that, use the settings in the "Retrieving users from LDAP" part to
	set the LDAP connection parameters.
